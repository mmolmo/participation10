Presenting a lovely Custom Font

This font was created with the help of Metaflop, a custom font creation site. The visuals of the font are changed by adjusting a row located at the left side of the site with many sliders, which control various properties.

For my font in particular, I chose to keep most of the settings the same as they were originally. Changes I made of note include pen width, which adjusts the line thickness of the typeface, unit width, which makes the letters slightly wider, some slanting, and a decreased taper.

These properties combined together create a font that is clean, visually appealing, and simple.
